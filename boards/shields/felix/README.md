#### Note to user:

- This shield is configured for a 4-row Felix Mini style layout (4x6+1 per half, 50 keys total).
- The active user keymap lives in `config/felix.keymap`, and the shield default keymap mirrors it for ZMK Studio compatibility.
- OLED (`nice_oled`) and nice!view e-paper (`nice_view_adapter nice_epaper`) builds are both included in `build.yaml`.
- `KANA` and `EISUU` are implemented with the `LANG1` and `LANG2` keycodes respectively.
- RGB underglow remains optional and is not enabled by default.
